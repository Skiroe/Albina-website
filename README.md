# Website AL Bina

De website wordt gehost via GitHub Pages. De broncode staat in dit publieke repository.

## Website overnemen
Indien het nodig is om de website te veranderen of over te nemen is het mogelijk om bewerkers rechten te krijgen op de repository. Dit kan via de eigenaar van de website (Nejmi). Die zal de orginele beheerder van de github repository kunnen bereiken.

Als alternatief is het ook mogelijk om de repository forken en daarop wijzigingen doorvoeren. 

## Info over formulieren
De contactformulieren gebruiken Formspree. Het e-mailadres waar de berichten naartoe gaan is gekoppeld aan de Formspree‑instelling.

### Wil je het ontvangende e-mailadres wijzigen? 
Dit kan doorgegeven worden aan de beheerder van de website. 

Indien dit niet mogelijk is: pas dan de Formspree‑configuratie aan door het vervangen van de `action`‑URL in `contact.html` en `contact-ar.html` met de nieuwe Formspree‑endpoint.
Hiervoor is een nieuwe Formspree account nodig, met eigen Formspree link. Dit duurt 5 min om aan te maken.
